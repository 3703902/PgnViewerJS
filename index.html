Overview PgnViewerJS

This is the raw documentation of work in progress, made available by GitHub Pages (which is great). You will find on this documentation the following:
  • A bunch of prepared documentation (overview, API, examples), which are available above in the menu (all menu entries except the one named "Home PgnViewerJS".
  • Some references on things you may be interested in like structure of this implementation, used plugins, additional examples, ...
If you have ideas how to improve the whole documentation, get in touch with me. Fork the repo, add issues, send me a pull request, ... whatever you want.

Behavior, Structure, Tests, Examples, ...

Here are the things that may help to get a better understanding what PgnViewerJS provides, how it is developed, and what may be reachable in a foreseeable future.
  • Testing: Testing of the faw functionality (nearly no UI) is done with Jasmine. See the file SpecRunner.html that should work out of the box. It shows a line of dots (hopefully all green), and you are able to check the status. If something goes wrong, just use the debugging tools of your browser.
  • Structure: Well, mostly grown. Some of it is understandable:
    • chess.js Base library from GitHub that knows chess inside-out.
    • chessboardjs Base library for building a chessboard from GitHub with excellent documentation.
    • css
    • docu: the "official" documentation, will be polished at some point in time
    • examples: the example directory (more below)
  • Examples: here comes it ...
    • board Shows different boards
    • comments games with a lot of comments (and faults)
    • edit shows how to edit games online (without the chance to store them)
    • variations some examples how to style the viewer
