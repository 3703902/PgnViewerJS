About PgnViewerJS?

PgnViewerJS was born as the third or fourth attempt to get nice PGN viewer for our chess club web site. PGN stands for Portable Game Notation which was born in the late 80s, early 90s of the last century. I provided some of the translations there ...

I first started using pgn4web and ... as part of our Wordpress Installation, but I had difficulties here and there. At the end, I wanted to have a viewer that was at least as nice as the one from chess.com

Implementation

The implememntation contains the following parts:

  • Chess.js: Small JavaScript library that provides the base functionality for playing chess (but without a board).
  • Chessboardjs: Builds the missing building blocks for displaying a chess board, with some functions: Moving pieces, some animations, ...
  • jQuery
  • jQuery Timer
  • jQuery Hotkeys
  • PgnViewerJS: builds the wrapper, makes the move output, allows some different interfaces: pgnBoard, pgnPrint, pgnView, pgnEdit
